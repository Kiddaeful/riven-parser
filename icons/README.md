# Icônes

Placez vos icônes dans ce dossier avec les dimensions suivantes :

- `icon16.png` - 16x16 pixels (icône dans la barre d'outils)
- `icon48.png` - 48x48 pixels (page de gestion des extensions)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Génération d'icônes

Vous pouvez créer vos icônes avec :
- Figma, Adobe Illustrator, ou Sketch
- Outils en ligne comme Canva
- Générateurs d'icônes en ligne

## Format

- Format : PNG avec transparence
- Couleurs : Utilisez le thème violet/bleu (#667eea, #764ba2) pour cohérence
